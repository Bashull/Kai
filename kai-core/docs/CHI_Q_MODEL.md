# CHI-Q MODEL

CHI-Q = puntuación ponderada de estado operativo del sistema:

- coherence (coherencia)
- heat (estrés)
- integration (unificación)
- queue (presión de pendientes)
- quality (confianza de extracción)
- risk (riesgo/sensibilidad)
- recovery (capacidad de restauración)

El valor final guía decisiones de continuar, pausar, pedir aprobación o priorizar memoria/auditoría.
