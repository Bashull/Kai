# MEMORY MODEL

KAI CORE usa 4 capas:

- **Episódica**: eventos de ejecución, fecha, fuente y resultado.
- **Semántica**: conceptos y relaciones estables.
- **Canónica**: decisiones finales y referencias maestras.
- **Operativa**: patrones, errores, soluciones y procedimientos.

La persistencia mínima se implementa en JSONL local; puede extenderse a vector store.
